Рекурсивний обхід директорії та виведення її структури.

Ця функція рекурсивно обходить дерево директорій, виводячи кожну директорію та файл з відступами для представлення структури.

Аргументи
pathПоточний шлях у дереві директорій.
output_fpВказівник на файл для запису, де буде записана структура.
levelПоточний рівень відступів (глибина у дереві директорій).
